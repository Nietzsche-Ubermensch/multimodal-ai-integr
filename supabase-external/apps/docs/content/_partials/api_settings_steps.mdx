Get your Project URL and API key from [the project's **Connect** dialog](/dashboard/project/\_?showConnect=true&connectTab={{ .tab }}&framework={{ .framework }}).

<Admonition type="note" title="API Key Types">

Supabase offers two key formats during the transition to improve security:

- **New keys** (recommended): Use publishable keys (`sb_publishable_xxx`) from the [**API Keys** tab](/dashboard/project/_/settings/api-keys/). Create one if needed by clicking the **Create new API Keys** button.
- **Legacy keys**: Use `anon` (client-side) or `service_role` (server-side) from the **Legacy API Keys** tab in the same [**API Keys** settings section](/dashboard/project/_/settings/api-keys/).

Both formats work during the transition. [Read more about the changes](https://github.com/orgs/supabase/discussions/29260).

</Admonition>

See the [API keys documentation](/docs/guides/api/api-keys) for detailed key usage.
