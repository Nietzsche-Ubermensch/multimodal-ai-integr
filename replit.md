# AI Suite - Multi-Provider AI Platform

## Overview
A React + TypeScript + Vite application that provides a multi-provider AI platform interface. Built with Radix UI components, Tailwind CSS, and various AI integrations.

## Project Structure
- `src/` - Main application source code
  - `components/` - React components (UI and feature components)
  - `lib/` - Utility functions and libraries
  - `hooks/` - Custom React hooks
  - `types/` - TypeScript type definitions
  - `data/` - Static data files
- `api-gateway/` - Optional API gateway service
- `supabase/` - Supabase configuration
- `docs/` - Project documentation

## Tech Stack
- **Frontend**: React 19, TypeScript, Vite 7
- **Styling**: Tailwind CSS 4, Radix UI Components
- **State Management**: TanStack React Query
- **Icons**: Phosphor Icons, Lucide React, Heroicons
- **AI Integrations**: OpenAI, OpenRouter

## Running the Project
The frontend runs on port 5000 using `npm run dev`.

## Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run test` - Run tests with Vitest
- `npm run lint` - Run ESLint
