## Quick Start

### 1. Discover patterns

```bash
npm run skill-master:discover
```

Shows what patterns exist in your codebase and which skills are missing.

### 2. Generate skills

```bash
npm run skill-master:generate
```

Creates SKILL files in `.claude/skills/` with real examples from your code.

### 3. Use with Claude

Simply mention pattern names when talking to Claude Code:

- "Create a new custom hook for..."
- "Add a service layer component..."
- "Build a UI component following our patterns..."

Claude will automatically use the appropriate SKILL file.
